Sistema operativo
Es el software encargado de administrar los recursos del computador, como la memoria, los archivos y los dispositivos. 
Permite que los programas funcionen correctamente y que el usuario interactúe con el sistema.

Node.js
Node.js es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. 
Se utiliza principalmente para desarrollar aplicaciones y servidores web de forma eficiente.

Aplicación server.js
Es el archivo principal que contiene el código del servidor en una aplicación Node.js. 
Su función es recibir solicitudes, procesarlas y enviar respuestas a los usuarios.

Puerto
Un puerto es un canal de comunicación que permite a las aplicaciones intercambiar datos a través de una red. 
Cada servicio utiliza un número de puerto específico para recibir conexiones.

Logs
Los logs son registros que almacenan eventos y actividades ocurridas dentro de una aplicación. 
Ayudan a monitorear el funcionamiento del sistema y a identificar posibles errores.
